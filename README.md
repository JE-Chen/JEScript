# JEScript

---

### Token

---

#### Variable

* var -> VAR_TOKEN

---

#### Data type

* int -> INTEGER_TOKEN

---

#### Symbol Operator

---

* :   -> COLON
* ;   -> SEMICOLON
* (   -> LEFT_PARENTHESES
* )   -> RIGHT_PARENTHESES
* {   -> LEFT_BRACES
* }   -> RIGHT_BRACES

---

#### Math Operator

* +   -> PLUS_TOKEN
* ++  -> PLUS_PLUS_TOKEN
* +=  -> PLUS_ASSIGN_TOKEN
* -   -> MINUS_TOKEN
* --  -> MINUS_MINUS_TOKEN
* -=  -> MINUS_ASSIGN_TOKEN
* %   -> MOD_TOKEN
* \*  -> MULT_TOKEN
* \>  -> GREATER_TOKEN
* \>= -> GREATER_EQUAL_TOKEN
* \<  -> LESS_TOKEN
* \<= -> LESS_EQUAL_TOKEN

---

#### Assign Operator

* = -> ASSIGN_TOKEN 

---

#### Compare Operator

* == -> EQUAL_TOKEN

---

#### Logic Operator

*  if             -> IF_TOKEN
*  else           -> ELSE_TOKEN
*  while          -> WHILE_TOKEN
*  BOOLEAN_TOKEN  -> true, false, TRUE, FALSE
*  AND_TOKEN      -> &&
*  OR_TOKEN       -> ||
*  NOT_TOKEN      -> !!
---

#### Comment Operator

* \#  -> LINE_COMMENT_TOKEN
* \## -> BLOCK_COMMENT_TOKEN

---

#### Bit Operator

---

#### Print

* print -> PRINT_TOKEN

---

#### Another

* undefined -> IDENTIFIER_TOKEN

#### End

* End -> EOS

---
